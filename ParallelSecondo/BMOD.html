Parallel BerlinMOD

Parallel BerlinMOD scales up the moving objects database benchmark BerlinMOD, making both the data generator and all OBA example queries run in Parallel SECONDO.

The complete benchmark can be downloaded through this link. It contains two parts, a parallel data generator and parallel OBA query scripts. Each parts contains a README file to explain the usage. A summary introduction is described in Example: How to Use the Parallel BerlinMOD Benchmark.

Base Data Files

Since SECONDO version 3.3, attribute names in SECONDO relations must start with a capital letter. Regarding this update, the BerlinMOD base data are also slightly changed. The adapted version can be downloaded from here.
